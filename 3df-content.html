3D FUNDAMENTALS

This module introduces students to the basic digital 3D production workflow to create assets for interactive projects. Students will practice basic modelling, UV unwrapping, digital sculpting, high-poly detail onto low-poly mapping, texturing, rigging, animation, real-time lighting, and rendering. Students will learn to produce hard-surface virtual objects for a real-time engine.

Topic 1: Model

Learn how to create 3D models using basic and advanced techniques.

Topic 2: Texture

Apply realistic textures and materials to enhance 3D models.

Topic 3: Rig

Learn how to create a skeletal system for 3D characters and objects.

Topic 4: Animate

Understand the principles of animation and how to apply them to 3D models.

Topic 5: Light

Learn how to set up real-time lighting and shadows for 3D scenes.

Topic 6: Render

Explore rendering techniques to achieve high-quality visuals.
