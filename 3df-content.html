3D FUNDAMENTALS

This module introduces students to the basic digital 3D production workflow to create assets for interactive projects. Students will practice basic modelling, UV unwrapping, digital sculpting, high-poly detail onto low-poly mapping, texturing, rigging, animation, real-time lighting, and rendering. Students will learn to produce hard-surface virtual objects for a real-time engine.

WK01: 3D TOOL FAMILARITY

Introduction to Maya.

WK02: 3D BLOCKING

Start working on ASG 1: Chess Piece by blocking the pieces out with simple shapes.

WK03: UV UNWRAP

Learn the core step before texturing the model.

WK04: PROCEDURAL TEXTURING

Learn the basics of Substance Painter.

WK05: BOOLEAN AND TOPOLOGY

Master the boolean function in Maya and make use of quad-draw to learn retopology.

WK06: TURNTABLE PRESENTATION SET-UP

Learn how to use Unity to set up a simple turntable to showcase your projects.
