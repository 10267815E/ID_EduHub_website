3D FUNDAMENTALS

This module introduces students to basic digital 3D production workflow to create assets for interactive projects. Students will practice basic modelling, UV unwrapping, digital sculpting, high-poly detail onto low-poly mapping, texturing, rigging, animation, real-time lighting and rendering. Students will learn to produce hard-surface virtual objects for real-time engine.

INTERACTIVE DEVELOPMENT

This module introduces students to front-end web development, covering HTML, CSS, and JavaScript to create interactive and responsive websites. Students will learn core web technologies, user interface design principles, and dynamic scripting to enhance user experience. Through hands-on projects, they will develop functional web applications while exploring modern frameworks and best practices.

PRODUCT MANAGEMENT

This module introduces students to agile product development, focusing on SCRUM methodologies and effective team communication. Students will learn to manage project workflows, prioritize tasks, and collaborate in cross-functional teams. Through real-world case studies and hands-on exercises, they will develop skills in sprint planning, backlog management, and iterative development to drive successful product outcomes.
